年终决策

发表于 2017-04-28 | 分类于 脑洞 | 阅读次数

异族的潘临收到教训后，最近几个月都老老实实的没敢来骚扰柴桑，我已下令，再来侵犯就咔嚓掉，为江东百姓除害，毕竟总对敌人仁慈也不好。

我最近除了不停的游说人才外，还要调整自己的状态来适应现在的生活。做为一个半吊子的穿越者，我还是得想办法怎样才能回去，这成为我内心深处最隐秘的秘密，所以我一面布局，一面处处留心，希望能找到可用的线索。

期间发生了一件叛乱的事情，让我大为恼火。是这样的，一名来自武陵的自称潘濬（jun，四声）的人来投奔我，我看他颇有学识的样子，于是邀请他在江陵城中做我的幕僚。考虑到我方现在处于发展期，所以只是给出了基本工资，让他去市场发展经济了。有一天他问我为什么不给他封官，我赶忙解释，过一段时间就可以封官了，现在我自己都没有官职，以后肯定会补偿你的云云，他没有说什么话，我以为是听进去了，没有在意。结果第二个月士兵向我汇报，说潘濬投奔了襄阳刘表！刘表给了个别驾的小官，他已经欣然接受了。我这个气啊，这个小子，也就这点出息了，刘表那渣渣我是丝毫不放在眼里的，告诉大家加强防备，以后等我们发展起来，看这个潘睿怎么哭着求我。

柴桑小捷

发表于 2017-04-28 | 分类于 脑洞 | 阅读次数

我方的四座城池肆意发展，终于惹来了敌对势力的针对。这年9月，军师来报，会稽的王朗因为有点小钱，看孔明各方面比自己强，羡慕嫉妒恨，于是请异族的潘临军团来柴桑捣乱。孔明准备请庞统过去，计划教训一下潘临一伙，我知道军师的超凡能力，就凭这些人肯定难不倒他，于是我立即派庞统过去，同时知会云长等人随时支援，我自己仍然坐镇江陵，招揽人才。

军师采用诱敌深入的策略，麻痹敌人，第一个月，他没有抵抗，潘临一伙在柴桑很是嚣张的祸乱了几天。等庞统一到，立刻派张飞等人带兵讨伐，可把张飞高兴坏了，他等了好几天了，讨伐的条件就是，听庞统的安排，原来军师仍然要驻守求贤所，他和庞统已经商量好对策了，准备好好教训下潘临。

兴点鸳鸯

发表于 2017-04-28 | 分类于 脑洞 | 阅读次数

晚上吃过晚饭，我出去探望了下子龙，得知子龙去军营训练新兵了，于是叫上关张和孔明，一块去军营巡视。这是“我”第一次看见我方的军士，子龙正在前方指挥操练，虽然是晚上但是毫不耽误。见到我来了，子龙让一名女将继续指挥，自己飞身过来。“主公怎么来了”，我微笑到，来看看你，还有大家。

我和众人走到训练的军士方阵面前，女将已停止指挥，欠身道，“拜见刘伯伯，拜见父亲，三伯伯，拜见军师”。我心里一乐，这不是关银屏吗，管他呢，不管真实的历史中有没有此人，这个世界有啊。的确是难得一见的美女。“不用客气，小关虎父无犬女啊，很有云长的风范。”，云长也是颇为自得，这个女儿对他来说，如同掌上明珠一样珍贵。云长无意间看到旁边的赵云有些拘谨，正自纳闷呢，我也看到了这个情况，一下就清楚了，我示意所有人安静，然后对军士们说了一些鼓励的话，并询问了一下军士的待遇情况，了解到大家吃的不是很好，于是我让军师记下来，明天升帐重点说明一下。然后让大家继续操练。
