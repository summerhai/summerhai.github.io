格局初现

发表于 2017-04-28 | 分类于 脑洞 | 阅读次数

虽然是子龙和关银屏的大婚第二天，但是局势紧张，次日还是继续升帐，此时大家的心情都是喜悦的，尤其是子龙和关银屏，似乎如胶似漆，虽然表面上看不出来，但感觉两个人的精神已经深深连在一起了。

我和孔明对望了一眼，我说道，“当前局势紧张，因此子龙和侄女的婚事太仓促了一些，你们二位见谅，以后肯定给补个好的”，子龙一跃道：“主公既已给子龙安家，末将必和银屏不离不弃，不求宏大婚事，只求报答主公大恩”，我微笑道，“子龙放心，听我和军师安排”。

军师此时已经打开了地图，平铺在中军帐的沙盘中，我走到中间，跟大家说到：“江陵城的经济基础比较好，人口流动较大，商贸也会随之发展，但是我们目前最大的问题就是，兵源不足”，我抬起头，拿起军令，“经过我和军师商量，我们需要扩充势力，众将听令：”，大帐中众将整装待发，等待我的命令下达。

“刘禅，你带领糜芳，张氏，糜氏以及5000人马，前往永安城发展经济，永安商贸发展繁荣，你要做的就是守住此城，发展经济，如果有敌人来攻，及时向江陵求援，接令！”刘禅冷不防一哆嗦，接令道：“谨遵父亲命令。”，说罢带着几人点兵离开了。

因为永安没有强力的外敌，我才放心让刘禅带队去的，“军师听令：命你带张飞，夏后氏，黄月英，并同5000人马，占领柴桑。柴桑地处要地，只有军师坐镇我才可以安心，你们的任务是发展兵力和粮食，并看一下是否能招揽优秀的人才，三弟，在外要听军师的命令，你们去吧”，说罢，把令交给了孔明，孔明心中早有安排，带着几人出去了。

光这两个城还不够，我继续吩咐道：“云长，你率领孙乾，赵云，关银屏以及5000人马，前往江夏招兵买马，同时大力发展科技，你和子龙现在关系非比寻常，一定要联手把事情办好”，一只令箭交到云长手里，云长欣然领命，和孙乾赵云等人离去。

此时大帐只有甘氏，简雍，糜竺，我对三人道，“非常时期，非常安排，夫人你也跟着辛苦下吧，大家和我一起发展江陵城，你们的主要任务是发展市场，我会全力招揽人才的”，我给众人打气，说实话，我之所以兵行险招，把兵力分散，是因为我发现城池之间可以相互支援，而且我周边除了刘表那个渣渣，没什么厉害的对手，所以率先把格局打开，坐等各方来信。
