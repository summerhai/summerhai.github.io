柴桑小捷

发表于 2017-04-28 | 分类于 脑洞 | 阅读次数

我方的四座城池肆意发展，终于惹来了敌对势力的针对。这年9月，军师来报，会稽的王朗因为有点小钱，看孔明各方面比自己强，羡慕嫉妒恨，于是请异族的潘临军团来柴桑捣乱。孔明准备请庞统过去，计划教训一下潘临一伙，我知道军师的超凡能力，就凭这些人肯定难不倒他，于是我立即派庞统过去，同时知会云长等人随时支援，我自己仍然坐镇江陵，招揽人才。

军师采用诱敌深入的策略，麻痹敌人，第一个月，他没有抵抗，潘临一伙在柴桑很是嚣张的祸乱了几天。等庞统一到，立刻派张飞等人带兵讨伐，可把张飞高兴坏了，他等了好几天了，讨伐的条件就是，听庞统的安排，原来军师仍然要驻守求贤所，他和庞统已经商量好对策了，准备好好教训下潘临。

张飞带齐柴桑所有的兵马，并发信让二哥带领子龙前来助阵，这样豪华的阵容打潘临真的好吗，我估计潘临的小脑袋都要不保了。云长那边收到消息后，带领子龙夫妇以及关兴，带足22000兵马前来助阵，那场面太美，我怕潘临会成渣。

我看到他们送来的布阵图的时候，一下就想到了军师的意图，很明显要活捉啊，原来孔明是看上了他们身上的宝贝，这下有的看了，虽然潘临手下普遍比较渣，但是有两个人居然有逆天的武器，他们可能不知道“怀璧其罪”的概念，但是遇到神出鬼没的孔明，只能自己说拜拜了。我在江陵城焦急的等待中，那边已经开打了。

潘临的优势在于他们的老家非常稳固，而且都适合在浅滩等作战，所以开始的时候他们跑得飞快，但是我方通过不断的诱敌出击，的确捕获了一名歹人，但是一名拥有神器的那个家伙居然在关键时刻跑了，怎么跑的，好像是突然就跑了，反正是没逮住，可把孔明气坏了，不过还好，已经抓到了一个，以后可以继续图谋另一个，因为他傻乎乎的还会跟着潘临干。

就这样，柴桑一役，俘获黄乱，没收他的神器莫邪宝剑，然后让他滚蛋了。我发出表彰，表彰柴桑和江夏第一次合作通力杀敌，取得良好效果。将神器存入宝库，等待封赏。
