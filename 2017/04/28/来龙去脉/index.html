来龙去脉

发表于 2017-04-28 | 分类于 脑洞 | 阅读次数

在众人的期盼中，孔明缓缓道，“话说汉高祖刘邦斩蛇起义，灭大秦，败项羽，开创大汉王朝，功绩无双。于是仙逝之后入驻天宫“圣皇殿”，这是历代皇帝的居所，是天帝为表彰他们为凡间统一建立的功业而设立的。昨夜我观看那流星，推算这应该是高祖刘邦感应到汉室纷乱，毅然舍弃安逸的天宫生活，化身流星来寻找世间最能拯救汉室危机的人。虽然流星的冲撞造成了一定的伤亡，但据我推算，现在主公应该是具有了“特殊”的能力，具体的能力要以后慢慢探索，总之，这是一件大好事”。孔明微笑的解释了一番。

但此时的我心里却震惊不已！刘邦？？圣皇殿？什么鬼？我之前的名字的确就是“刘邦”啊！！难道我曾经上过天？还会变流星？这一头雾水让我一时班会不知道说什么，我把张开的嘴闭上，先冷静一下，想想怎么回事。此时，云长起身问道，“这样说来，大哥以后会洪福齐天，成就霸业了？”，听到这里，我回了回神，看孔明怎么回答。

“并不尽然，还是需要我们一起辅佐，但是高祖的护佑会提高主公的个人魅力，也就是说主公可以很轻易的招揽人才，在这个时代，人才才是最重要的，主公你说对吗？”。我点了点头，丫的刘备本身自带的光环，招人就是犀利啊。“这次的事件不要轻易外传，以防有人联合起来对付我们。对了，现在身体已无大碍，军师可以说下我们的目前情况”。我觉得有点懂了，似懂非懂，不过这个慢慢再研究，先搞清楚，这里是不是三国志12里的那个世界。

“主公，现在我们屯兵江陵，此地物产丰富，人杰地灵，据我推算，很多人才都会慕名而来的，主公要多费心了。至于军事方面，关张赵等将军在军营的号召力无与伦比，现在共有军士20000,。粮草方面，由于江陵城小，目前只存了23000担军粮，需要进一步提升”，孔明顿了一顿，“周边有几个城市还没有人占领，正想请教主公，关于大局方面，我们是不是要开始准备了？”。

我会心一笑，“正等和军师商量呢”。
