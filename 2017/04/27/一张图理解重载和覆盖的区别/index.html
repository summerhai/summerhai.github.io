覆盖和重载是Java中比较重要的两个概念，对初学者来说经常会感觉到困惑，本文用简单的例子解释二者的区别。

定义

方法签名：方法的名字和参数列表
重载：发生于同类方法中，方法名相同、方法签名不同
覆盖：发生于继承的父类和子类中，方法名和方法签名相同，但是父类和子类对该方法的实现不同，子类的实现允许和父类的实现不同。

覆盖与重载

有以下几个重要情况需要注意：
1， 在运行时的对象类型决定了哪个覆盖方法该被使用，而引用变量类型决定了哪个覆盖方法将被用于编译期。（这话看起来好怪，放原文吧：The real object type in the run-time, not the reference variable’s type, determines which overridden method is used at runtime. In contrast, reference type determines which overloaded method will be used at compile time.）
2， 覆盖体现多样性，而重载不能。
3， 覆盖是运行时的概念，而重载是编译时的概念。
